Changelog:

1. Basic modification configurations have been implemented for a relatively stable game.

2. Main Ru/Eng GUI has been created (outside of gameplay).

3. Optimization has been tested in the main world on an medium PC. (Stable 140+ FPS, falling to 90-120 in areas with high amount of entities)

4. Added starter kit.

5. LAN multiplayer has been optimized. (Or rAdmin/Hamachi)

6. Texture packs have been changed.

7. Custom textures for glass and glass panels have been drawn.

8. Preliminary!! launch parameters have been set. (Available when installing via CurseForge or when manually added to parameters).
-Xms10G -Xmx10G -XX:+UseG1GC -XX:+UnlockExperimentalVMOptions -XX:+DisableExplicitGC -XX:+ParallelRefProcEnabled -XX:MaxGCPauseMillis=150 -XX:InitiatingHeapOccupancyPercent=50 -XX:G1NewSizePercent=20 -XX:G1MaxNewSizePercent=40 -XX:G1HeapRegionSize=4M -XX:G1ReservePercent=10 -XX:SurvivorRatio=32 -XX:MaxTenuringThreshold=1 -XX:ParallelGCThreads=10 -XX:ConcGCThreads=6

9. Preliminary keybind controls have been set.

10. Basic player parameters have been changed: regeneration, attack strength, block breaking speed (including with tools).

11. Bad Omen can now be removed with milk. (May be removed in the future)

12. After sleeping, all effects will be removed, and a portion of health will be restored.

13. Shader packs have been added (without configurations).

14. Stat boosts have been temporarily disabled (will return in version 0.3.x).

15. The crafting of certain items has been removed. (Currently, they are replaced by recipes from other modifications)

16. After launching the game world: territory loading automatically begins (a 500-block square from the player's spawn point), a welcome message appears, a menu with important functions opens (FAQ, project links, territory preloading (chunk-preload 100/250/500/1000(not recommend) blocks in the Overworld, Nether, End, Bee World, and other), driver reload (due to possible conflicts between Embeddium and other optimization mods)).

17. Log message spam from killing villagers has been disabled (for optimization purposes).

18. Some blocks (like the stonecutter) now deal damage to the player.

19. The mechanics of torches, campfires, furnaces, and similar items have been changed.

20. Sitting animations have been added. (Animations vary depending on the block the character is on)

21. The behavior logic of all classic mobs has been changed.

22. The hunger, thirst, fatigue, and temperature systems have been optimized.

23. Some FTB modules have been disabled.

24. The Overworld, caves, and dimensions now feature entirely new ambient and another sounds.

25. World generation has been changed.

26. Villagers no longer take damage from berry bushes.

27. Shield characteristics have been changed. New ones have been added.

28. Base values of some items in mods like Botania, Minecolonies, and Delightfull have been changed.

29. Climbing has been added.

30. Some relics from the Relics mod have been removed. (conflict/disrupting the main concept of the pack)

31. The experience gained by equipment has been changed. (Items now gain experience much faster. Previously, even iron items had trouble reaching level 5.)

32. Changed the logic of the horde that appears every few days. (Approximately every 10-25 in-game days)

33. Raid can now start regardless of the presence of Bad Omen.

34. Experimental features have been disabled during world creation.

35. The player can no longer select the difficulty and certain gamerules.
